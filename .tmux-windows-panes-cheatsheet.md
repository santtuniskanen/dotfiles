# tmux Windows & Panes Cheatsheet (Prefix: Ctrl+a)

## Windows
- New window:           Prefix + c
- Next window:          Prefix + n
- Previous window:      Prefix + p
- Select window by #:   Prefix + [0-9]
- List windows:         Prefix + w
- Rename window:        Prefix + ,
- Close window:         Prefix + &

## Panes
- Split vertically:     Prefix + "
- Split horizontally:   Prefix + %
- Switch pane:          Prefix + Arrow Key
- Cycle panes:          Prefix + o
- Swap panes:           Prefix + {
                        Prefix + }
- Kill pane:            Prefix + x

## Resizing Panes (if configured)
- Resize left:          Prefix + Shift + Left
- Resize right:         Prefix + Shift + Right
- Resize up:            Prefix + Shift + Up
- Resize down:          Prefix + Shift + Down

## Misc
- Show key bindings:    Prefix + ?
- Reload config:        Prefix + r (as in your config)
